# MAC Address

## What is a MAC Address?

A MAC (Media Access Control) Address is a unique hardware identifier assigned to a device's network interface card (NIC). It helps devices identify each other on a local network.

Every device connected to a network has a MAC address, such as a computer, smartphone, printer, or gaming console.

---

## Why MAC Addresses Matter

MAC addresses allow devices to communicate within a local network. They help switches determine where data should be sent and ensure that information reaches the correct device.

Without MAC addresses, local network communication would not function properly.

---

## MAC Address Format

A MAC address is a 48-bit address written in hexadecimal format.

### Example

```text
A1:B2:C3:D4:E5:F6
```

Another common format:

```text
A1-B2-C3-D4-E5-F6
```

A MAC address contains numbers and letters from 0-9 and A-F.

---

## MAC Address vs IP Address

| MAC Address            | IP Address           |
| ---------------------- | -------------------- |
| Physical Address       | Logical Address      |
| Layer 2                | Layer 3              |
| Usually Permanent      | Can Change           |
| Used on Local Networks | Used Across Networks |

### Simple Way to Remember

Think of a MAC address as a person's fingerprint.

Think of an IP address as their current home address.

---

## Real-World Example

When a laptop sends data to another device on the same network, the switch uses MAC addresses to determine which device should receive the data.

For example:

```text
Laptop MAC:
A1:B2:C3:D4:E5:F6

Printer MAC:
F6:E5:D4:C3:B2:A1
```

The switch learns these addresses and forwards traffic correctly.

---

## Cybersecurity Connection

MAC addresses are important in cybersecurity because analysts use them to identify devices on a network.

They can help:

* Track devices
* Investigate suspicious activity
* Analyze packet captures
* Understand network traffic

Tools such as Wireshark often display MAC addresses during packet analysis.

## Commands to Practice

### Windows

```bash
getmac
```

Displays the MAC addresses of your device.

```bash
ipconfig /all
```

Displays network configuration information, including MAC addresses.

```bash
arp -a
```

Displays the ARP table and learned MAC addresses.

## Practice Questions & Answers

### 1. What does MAC stand for?

 Media Access Control.

### 2. What is a MAC address?

 A unique hardware address assigned to a network interface card.

### 3. What OSI layer uses MAC addresses?

 Layer 2 (Data Link Layer).

### 4. Which device uses MAC addresses to forward traffic?

 A switch.

### 5. Is a MAC address physical or logical?

 Physical.

### 6. Can an IP address change?

 Yes.

### 7. Does a MAC address usually change?

 No, it is generally tied to the network interface.

They help devices identify and communicate with each other on a local network.

getmac
